This is a basic HTML project built as part of The Odin Project curriculum.
The goal of this project was to practice creating a simple multi-page website using only HTML.

The site contains a homepage that links to individual recipe pages.

⸻

Through this project, I practiced:

* Structuring a basic HTML document
* Using headings, paragraphs, and lists
* Creating links between pages (<a> tags)
* Organizing files and folders in a project structure
* Understanding how multi-page websites work

⸻

Project Structure
/recipes-project
  ├── index.html
  ├── recipes/
  │     ├── recipe1.html
  │     ├── recipe2.html
  │     └── recipe3.html

⸻
   How to View

1. Clone or download the repository
2. Open index.html in your browser
3. Click through the recipe links
